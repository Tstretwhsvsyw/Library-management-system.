# Library Management System

A Java-based Library Management System developed using Object-Oriented Programming and layered architecture.

## Features
- Add new books
- View available books
- Issue books to users
- Menu-driven console interface

## Technologies Used
- Java
- IntelliJ IDEA
- OOP (Encapsulation, Abstraction)
- Collections Framework

## Project Structure
- model – Book entity
- dao – In-memory data storage
- service – Business logic
- util – Input handling
- main – Application entry point

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Run `Main.java`
4. Use the console menu
